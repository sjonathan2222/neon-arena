# Neon Arena v2.0

A fast-paced LAN multiplayer shooter built with Node.js and Socket.io. Features classes, skills, and a co-op mode.

## Installation

1. Install Node.js.
2. Open terminal in this folder.
3. Run `npm install` to install dependencies.
4. Run `npm start` or `node server.js` to start the server.

## Playing

* **PC:** Open `http://localhost:3000`. Use WASD to move, Mouse to aim/shoot, SPACE to dash.
* **Mobile:** Connect to the host's IP (e.g., `http://192.168.1.5:3000`). Use on-screen joysticks.
    * **Left Stick:** Move
    * **Right Stick:** Aim & Shoot
    * **Button:** Dash
